Free, Libre, and Open Source Software for Building Automation

About the Initiative

The BAClib Initiative is a primarily recreational effort focused on building automation and control, providing free, libre, and open source software for the broader community. Drawing on decades of experience with BACnet and related protocols, the initiative aims to create modern, well-designed tools and libraries that benefit the entire building automation ecosystem, with a strong focus on workflows across the full project lifecycle, including planning, engineering, implementation, realization, commissioning, and operation.

As a volunteer-driven project developed in contributors' free time, progress follows the natural pace of recreational development rather than commercial timelines.

Projects

BACnet ASN.1 Parser

A lightweight Node.js parser for BACnet ASN.1 data type definitions. It converts ASN.1 declarations into structured objects and normalizes the output into BAClib-compliant kebab-case formats.

Supports key BACnet constructs including ENUMERATED, SEQUENCE, CHOICE, BIT STRING, OCTET STRING, APPLICATION tags, constraints, and extensibility markers.

Generic BACnet Types

A language-agnostic BACnet type system based on JSON definitions, designed to enable consistent code generation and tooling across multiple programming languages.

The project roadmap includes generators for platforms such as C#, C/C++, Rust, and JavaScript, plus support for documentation and interface generation from shared type definitions.
